Install Google Antigravity
Installs the Antigravity CLI, all Linux dependencies, and configures ChromeOS browser integration. Run once.
Open Terminal → paste this:
curl -fsSL https://leftdisplay.com/chromebook/scripts/install-antigravity.sh | bash
1 Open the Terminal app on your Chromebook
2 Paste the command above and hit Enter
3 When done, type agy to launch Antigravity
▶️
Step 2: Launch Antigravity
Run this AFTER install completes. Refreshes your PATH and launches Antigravity.
Paste into Terminal:
source ~/.bashrc && agy
🔍
Troubleshoot: Run Diagnostic
If "agy" says "command not found", run this. It scans your system and shows where everything installed.
Paste into Terminal:
curl -fsSL https://leftdisplay.com/chromebook/scripts/diagnostic.sh | bash
🛠️
Fallback: Run by Full Path
If nothing else works, try launching directly by full path.
Paste into Terminal:
~/.local/bin/agy
🖥️
Plan B: SSH Into Your Mac
If "illegal instruction" error — your Chromebook CPU can't run the binary. Instead, use your Chromebook as a remote terminal into your Mac where Antigravity already works.
Paste into Terminal:
ssh jeffkerr@192.168.1.138
Enter your Mac password when prompted. Then type agy to run Antigravity through your Mac. Both machines must be on the same WiFi.
💡
Tip: Paste in Terminal
On Chromebook Terminal, paste is Ctrl + Shift + V (not just Ctrl+V).
