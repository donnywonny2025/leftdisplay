# LeftDisplay Hosting Framework

This repository is the dedicated root folder for the `leftdisplay.com` web hosting framework. 

**Purpose:** 
To separate public-facing demonstrations, websites, and portal subsets from private backend project repositories (like DAAD). 

**Deployment Strategy:**
1. This root directory maps directly to `leftdisplay.com` via Netlify.
2. Any project you want to publish should be added to this repository as a subfolder (e.g. `leftdisplay.com/your-project/`).
3. Keep private knowledgebases and legal data OUT of this repository unless specifically intended to be public.
